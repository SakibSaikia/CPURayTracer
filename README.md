CPU path tracer based on Perter Shirley's [**Ray Tracing in One Weekend**](https://www.amazon.ca/Ray-Tracing-Weekend-Minibooks-Book-ebook/dp/B01B5AODD8). 

* Implemented in C++17
* Direct2D for live preview. 
* Multithreading
* DirectXMath for SIMD
